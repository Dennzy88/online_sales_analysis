#  Online Sales Analysis

This is a Python-based project for managing product inventory and shopping cart functionality in an online store scenario. It also demonstrates version control with Git and GitHub.

##  Features

- Add, display, update, and remove products
- Manage inventory value
- Add items to a customer cart
- Calculate total cart value
- Use of Git branching, merging, and conflict resolution
- Versioned and documented with GitHub

##  Technologies

- Python 3
- Git & GitHub

##  Project Structure

```
src/ → Python modules (Product, ProductManager, Cart)
main.py → Entry point of the application
screenshots/ → Git status screenshots per task step
.gitignore → Excludes sensitive files like config.json and images
README.md → Project documentation
notes.txt → Task summary and repository link
```

##  How to Run

Make sure you have Python 3 installed. Then run:

```bash
python main.py
```


##  Security
Do not share or push your config.json file — it contains sensitive keys and data.


##  GitHub Workflow

- Multiple branches are used for each feature:
  - `add-product-removal`
  - `add-cart-functionality`
- Each branch is merged into `main` with conflict resolution when necessary.
- Clear commit messages are provided for each step.
